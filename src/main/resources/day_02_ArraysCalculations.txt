Arrays - codul se scrie itro clasa noua in package arrays,
si se cheama din clasa ArraysCalculations, fiecare in metoda cu varianta lui

De facut un array unidimensional mai numit si vector,
de introdus numarul de elemente de la tastatura (se poate de folosit clasa Scanner)
de populat elementele tabloului cu numere negative si pozitive utilizind clasa Random.

Vedeti varianta fiecaruia conform instructiunii din variantele.txt
Varianta_1: calculeaza suma, produsul, media aritmetica si calculeaza formula:
produs * suma / media

Varianta_2: calculeaza min si max si calculeaza formula:
min * max / nr_elem

Varianta_3: extrage numerele pozitive si negative in doua arrayy-uri (sau List) separate si calculeaza suma modulelor lor:
|suma(array_pozitive)| + |suma(array_negative)|

Bonus 1:
calculeaza suma numerelor negative si produsul numerelor pozitive si calculeaza restul impartitii intre produs si suma.
restul ( produs(array_negative) / suma(array_pozitive) )

Bonus 2:
De creat un array cu numere float.  De extras partea intreaga si partea flotanta in doua array-uri separate.
De calculat sumele la fiecare si suma totala.
De comparat rezultatul obtinut cu suma array-ului original.
